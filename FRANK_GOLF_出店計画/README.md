# FRANK GOLF 事業計画（このフォルダについて）

このフォルダは **FRANK GOLF の事業計画**です。

> ⚠ **注意：これは「GOLF WING 2号店」ではありません。** FRANK GOLF は独立した事業計画です。GOLF WING の2号店出店計画は別フォルダ `GOLFWING_2号店_出店計画/` を参照してください。
>
> （初版作成時にタイトル等へ「GOLF WING 2号店」表記が残っている箇所がありますが、本計画は FRANK GOLF として扱います。数値・スケジュールはそのまま有効です。）

## 収録物
- `00_計画サマリー_必要資料と前提.md` — 必要資料/不足情報/仮置き前提
- `01_出店スケジュール.md` / `02_ガントチャート.csv` — 工程
- `03_FRANK_GOLF収支計画.xlsx`（＋`_build.py`） — 4シナリオ数式連動モデル（居抜き/2打席/省人モデル前提）
- `04〜08` — ファイン承諾資料 / 金融機関サマリー / リスク一覧 / 確認事項 / アジェンダ
- `09_仕組み化提案.md` / `templates/` — 資料自動生成の仕組み
