## 3. 仮置き前提一覧（すべて「仮置き」。根拠と保守補正を明記）

### 3-1. 事業前提
- **業態**：姫路の会員制インドアゴルフ（GOLF WING 2号店 = FRANK GOLF）。2打席想定、レッスン＋無人運営＋システム化中心の省人モデル。〔仮置き〕
- **会員上限**：200名（宝塚が210名到達の実績より上限を200で保守設定）。〔仮置き〕
- **平均月会費（税抜）**：標準15,000円。宝塚の在籍加重平均は約18,800円（税抜）だが、法人・マスター比率が高く新店では再現困難とみて**保守側に約2割ディスカウント**。〔仮置き／根拠: 宝塚会員種別×単価実績〕
- **入会金**：33,000円（税込）→税抜30,000円。〔仮置き〕
- **体験単価**：2,200円（税込）。〔仮置き〕

### 3-2. 会員動態（数式で自動連動）
- **入会数** = 体験数 × 入会率（capacity上限あり）
- **退会数** = 期首会員 × 退会率
- **退会率**：標準4% / 悲観6% / 最悪8%（月次）。宝塚実績は月3.5〜3.9%だが新店は定着弱く保守。〔仮置き〕
- **入会率**：標準35% / 悲観25% / 最悪18%。宝塚は月により53〜128%だが体験非経由入会を含む特殊値のため、新店は**体験→入会の純転換のみ**を保守設定。〔仮置き〕
- **1年目集客遅れ**：体験数はY1を抑え、Y2・Y3で回復（Assumptions参照）。〔仮置き〕

### 3-3. コスト前提（月次・OPEX）
| 項目 | 標準（円/月） | 根拠・補正 |
|---|---|---|
| 家賃 | 400,000 | 既存資料。姫路相場で仮置き |
| 人件費 | 580,000 | 既存資料。無人化前提でも受付/レッスン最低限。最悪は増額 |
| 光熱費 | 80,000 | 既存資料。空調負荷で変動 |
| 広告費 | 80,000（Y1は120,000） | 既存資料。**初年度は集客投資で増額が現実的** |
| 通信費 | 15,000 | 仮置き |
| 保険 | 12,000 | 施設賠償・動産 |
| システム費 | 40,000 | 予約/会員/シミュレーター月額。ファイン条件で変動 |
| 清掃費 | 20,000 | 無人運営は清掃外注が要 |
| 修繕・設備更新積立 | 30,000 | **既存楽観案に欠落**。機材故障備えで計上 |
| 消耗品費 | 15,000 | 仮置き |
| 税理士費用 | 30,000 | **既存楽観案に欠落**。月次顧問 |

### 3-4. 初期投資（CAPEX）— **既存10,340,000円は保守再検証で増額の可能性大**
- 既存「初期投資約1,034万円」には**保証金・予備費・開業前費用が含まれていない疑い**。本モデルでは保証金・予備費を明示し、**総所要資金＝CAPEX＋保証金＋運転資金**で提示。
- 結論（詳細は`06`/Excel CAPEX）：**総所要資金は出資想定1,500万円を上回る可能性**があり、運転資金の薄さが最大リスク。→ 追加融資枠 or CAPEX圧縮の必要性を先回りで提示。〔仮置き〕
