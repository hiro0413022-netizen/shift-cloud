# 収支計画テンプレート（11シート）

| シート | 内容 |
|---|---|
| Summary | 売上/営業利益/現金残高/損益分岐点/投資回収/資金ショート月数（経営判断用） |
| Assumptions | 全入力の集約。会員数/会費/入会金/体験単価/入会率/退会率/広告/人件費/家賃/初期投資等。他シートはここを参照 |
| Scenarios | 楽観/標準/悲観/最悪を横並び。選択セル＋INDEX/MATCHで自動反映 |
| Monthly_PL | 36ヶ月。会員動態→売上→変動費→固定費→営業/税前/税引後利益 |
| Cash_Flow | 初期投資/運転資金/月次営業CF/累計CF/月末現金。ショート月は赤 |
| BreakEven | 損益分岐点会員数/必要月商/必要体験数/必要入会数 |
| CAPEX | 内装/設備/シミュレーター/PC/NW/防犯/備品/看板/広告初期/保証金/予備費 |
| OPEX | 家賃/人件費/広告/光熱/通信/保険/システム/清掃/修繕/消耗品/税理士/決済手数料/予備費 |
| Sensitivity | 会員数/会費/退会率/広告/人件費/家賃変動の利益インパクト |
| Risks | 出店/集客/運営/資金繰り/人材/設備/法務/契約 × 影響度/可能性/対策 |
| Investor_QA | 想定質問と回答案 |

ルールは `Excelモデル作成ルール.md` を参照。
