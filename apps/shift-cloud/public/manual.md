# RUNBOOK — シフトアプリ（Shift Cloud）現場手順書

対象: 店舗スタッフ（打刻・日報・シフト提出）と店舗責任者（勤怠修正・給与前チェック）。
URL: https://shift-cloud-shift-cloud.vercel.app （スマホのホーム画面に登録しておくと楽です）

---

## A. スタッフ: iPadで打刻する

1. 受付のiPad（打刻キオスク）で自分の名前を選ぶ
2. 【出勤】【退勤】【休憩開始】【休憩終了】のどれかを押す — これだけです
3. **押し忘れた/間違えたら**: キオスクの「連絡・打刻忘れの報告」から報告を入れる（→責任者が後で直します）。口頭でもOK

**休憩について**: 休憩打刻を押さなくても、法律どおりの休憩（6時間超=45分/8時間超=60分）が自動で引かれます。実際の休憩が違った日は責任者に伝えてください。

**残業について**: 退勤が数分過ぎただけでは残業になりません。**退勤打刻を15分単位に切り下げてから**シフト終了と比べます（例: 終了19:45・退勤19:54 → 残業0）。遅刻・早退・実働時間は打刻そのままです。

## B. スタッフ: 毎日のアプリの使い方（スマホ）

下のメニューで移動します。**ホーム → 日報**の2つを押さえておけば大丈夫です。

| メニュー | やること |
|---------|---------|
| 🏠 ホーム | **今日のシフト・今日のやること・お知らせ**。まずここを見る |
| 📅 カレンダー | 自分の予定＋店舗共通の予定（予約申込など）を月表示で |
| ✋ シフト提出 | 次回シフトの希望を出す。**出勤募集**（人が足りない日）への応募もここ |
| 📝 日報 | 今日の日報・今週の週報を書く。**みんなのレポート**で他メンバーの分も読める |
| 🔔 お知らせ | 会社・店舗からの連絡。既読にする |
| 💬 聞く | 売上・会員・体験予約・シフト・勤怠に日本語で質問（見えるのは**自店舗**まで。給与・経理・契約は見えません） |

**「今日のやること」について**

- 個人あてのタスクと、**「店」バッジの店舗共通タスク**（Reserve OSの予約申込など）が並びます
- 店舗共通は**誰か1人がチェックすれば全員から消えます**。取り合いにならないよう、やる前にチェックせず、やってからチェック
- 予約申込は【詳細】で希望日時・連絡先が見られます（そのまま折り返し電話へ）
- 自分用のメモは「やることを追加」で足せます

## C. 責任者: 打刻を修正する

1. PCでログイン → 管理画面 →「勤怠管理」
2. 対象スタッフ・日付の行の【修正】を押す
3. 直すところだけ入力する:
   - 出勤/退勤の時刻
   - **休憩（分）** … 実際の休憩時間に上書きできます。「自動計算に戻す」にチェックすると自動値に戻ります
   - **修正理由**（必須）… 例:「打刻漏れ／休憩の実態に合わせて修正」
4. 保存すると行に「修正済」と表示されます（元の打刻は消えず履歴に残ります＝改ざん防止 DECISIONS #6）

## D. 責任者: 打刻忘れ報告を確認する

1. 管理画面 →「打刻端末メモ」
2. 「未対応」タブの報告を確認 → Cの手順で勤怠を直す → 報告を「対応済み」にする
3. **月末照合の前に未対応をゼロに**しておくと給与計算がスムーズです

## E. 月末: 給与計算の前チェック（責任者）

1. 勤怠管理で「打刻漏れ」フラグの日が残っていないか確認（シフトがあるのに出勤/退勤が無い日）
2. 打刻端末メモの未対応をゼロに
3. 給与画面は追加のパスワード確認があります（給与データ保護のため）

## F. アカウント・権限

- ログインID・パスワード・役割は **GENESISの「アカウント管理」** または Shift Cloud 管理画面 →「スタッフ」で古川さんが設定します（データは共通・どちらでも同じ）
- パスワードを忘れた・入れない → 古川さんへ（再発行できます）

## 困ったとき

| 症状 | 対処 |
|------|------|
| キオスクに自分の名前がない | 店舗の配属設定漏れ。責任者→古川さんへ |
| 二重打刻してしまった | そのままでOK。責任者が修正で直せます |
| 残業がついていない / 少ない | 15分の切り下げルールのためかもしれません（上記A）。実態と違うなら責任者へ |
| 「この月は締め済みです」と出る | 締め後の修正は古川さんへ（締め解除が必要） |
| 時給が月の途中で変わった | 何もしなくてOK。**変更日から自動で新時給**で計算されます（変更前の日は旧時給のまま） |
| 「やること」が急に消えた | 店舗共通タスクを他の人が完了した可能性。カレンダー or 店長に確認 |
| 「聞く」で給与のことを聞いても答えが出ない | 仕様です。スタッフからは給与・経理・契約は見えません（本部のみ） |
