# RUNBOOK — レッスンカルテ（Lesson OS）現場手順書

対象: レッスンを担当するコーチ。
URL: （デプロイ後にVaultとここへ記載）

---

## 1. ログインする

- Shift Cloudと同じID・パスワード
- 「アクセス権がありません」→ レッスン権限（use_lesson）が未付与。古川さんへ

## 2. レッスンの流れ（1人あたり1分で終わる記録）

1. 生徒一覧から生徒をタップ（いなければ【＋生徒を追加】— **名前だけでOK**）
2. カルテの「スイング動画を追加」→ ファイル選択（**スマホ・タブレットならその場でカメラ撮影**）
3. クラブ（DR/7I等）とひとことメモを入れて【⬆アップロード】
4. 動画の下の欄に**コーチコメント**を書いて送信（あとから何度でも追記OK）
5. 良いスイングが撮れたら【☆ベストにする】（生徒ごとに1本、次回の見本になります）

## 3. 目標の使い方

- カルテ上部の【✎編集】で生徒の目標（例:「90台で回る」）とメモ（体の特徴・注意点）を設定
- 目標は一覧にも出ます。**レッスンの最初に目標を口に出して確認**すると継続率が上がります

## 4. してはいけないこと

- 動画を**生徒本人以外に見せない・SNSに上げない**（個人情報です。本人への共有機能は準備中）
- 動画の削除は間違えて撮った場合のみ（上達の記録が消えます）

## 困ったとき

| 症状 | 対処 |
|------|------|
| アップロードが終わらない | 電波を確認。200MB超の動画は分割 or 短く撮り直し |
| 動画が再生できない | 再読み込みして再度【▶再生】（再生リンクは10分で切れる仕様） |
| 生徒を間違えて登録した | 古川さんへ（統合・削除できます） |
