# YOZAN GENESIS — 役員向け共有パック（小川うららさん用）

作成: 2026-07-13。使い方の手順は「使い方マニュアル」（ログイン画面の📖 / RUNBOOK.md）とセットで読んでください。

---

## 1. このシステムは何か・何ができるか

GENESISは**会社全体の司令室**です。GOLF WING・FRUNK GOLF・KALLINOS・キャディ事業のデータが毎日自動で集まり、AIが分析して「今日、経営が判断すべきこと」を出します。

できること（役員目線）:

- 会社の総合スコア・売上・利益・会員数・人件費率を**開いた瞬間に見る**
- AIが毎朝つくる**日次レポート**を読む（何が起きたか・何が危ないか・何をすべきか）
- AIや現場からの**承認依頼に答える**（承認/却下ボタンだけ）
- お客様からの**問い合わせとAIの返信案を確認**して送信を許可する
- **資料室**で出店計画などの資料をダウンロードする
- **社内連絡**で古川さんに伝えたいことを書き残す（口頭やLINEで流れがちな連絡をここに）

## 2. 何をしたら、どこに入力するか（業務→入力先の対応表）

| したこと・起きたこと | 入力する場所 | その結果 |
|---|---|---|
| 出勤・退勤 | 店舗iPadで打刻（Shift Cloud） | 勤怠→人件費KPIに自動反映 |
| シフトの希望を出す | Shift Cloudスタッフ画面「シフト提出」 | 店長が確定→カレンダーに反映 |
| 体験のお客様が来た | 受付タブレット（Member OS） | 体験予約数・入会率KPIに自動反映 |
| 入会が決まった | Member OSで【入会】ボタン | 会員数KPIに自動反映 |
| 契約書が届いた | Legal OSに登録＋PDF添付 | 期限が近づくと自動で判断リストに出る |
| 領収書・請求書を受け取った | Money OS /receipts に撮って登録 | 翌朝AIが日付・金額を自動読取り |
| 会社の資料を共有したい | GENESIS「資料室」にアップロード | 権限者がどこからでもDL可能 |
| 古川さんに伝えたいこと | GENESIS「社内連絡」に記入 | 古川さんの画面に未対応として表示 |
| 日々の気づき・報告 | Shift Cloudスタッフ画面「日報」 | 社内で共有される |

**原則: 同じことを2か所に入力しない。**入力は現場アプリに1回だけ、GENESISは見る場所です。

## 3. 何がどのスケジュールで自動で動くか

| タイミング | 動くもの | 内容 |
|---|---|---|
| 毎朝6:00 | CEO AI日次レポート | KPI再集計→スコア算出→「今日判断すべきこと」生成 |
| 毎朝6:00（同時） | KPI整合性チェッカー | 経費未入力・目標未設定など「数字の嘘」を検知して警告 |
| 毎朝6:00（同時） | 法務チェック | 契約の期限接近・高リスク・滞留を判断リストへ |
| 毎朝6:00（同時） | 法務AI読取り | 未読取りの契約書を1日1件、AIが要点抽出（提案として保存） |
| 毎朝6:00（同時） | 経理AI読取り | 未読取りの領収書を1日3件、日付・金額を自動補完 |
| 給与: 毎月25日前後 | （手動）給与計算 | Shift Cloud管理画面で締め→明細と1円一致の検証済みロジック |
| 随時 | KPI更新ボタン | 「CEO AI 司令室」から手動でも再集計できる |

※資料室に自動スケジュールはありません（アップロード・ダウンロードとも手動、いつでも可能）。

## 4. 小川さんの権限でできること・できないこと

- できる: 全タブの閲覧 / 承認待ちへの回答 / 資料室のアップロード・DL / 社内連絡の記入
- できない: スタッフの権限変更・給与画面（Shift Cloud管理系）→ 古川さんの権限
- ログインID・初期パスワードは古川さんから受け取ってください（Shift Cloudと共通）

## 5. 最初の1週間のおすすめ

1. 毎朝「経営ダッシュボード」を開いて判断リストを読む（3分）
2. わからない用語・数字があったら「社内連絡」に書く（改善のタネになります）
3. 資料室の出店計画に目を通す
