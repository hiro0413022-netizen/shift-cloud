# RUNBOOK — 体験受付・会員（member-os）現場手順書

対象: GOLF WING 受付スタッフ（PCに詳しくなくてOK）。FRANK GOLF 姫路の担当者は §7・§8 も。
URL: https://member-os-tau.vercel.app （ホーム画面にアイコン登録しておくと楽です）

---

## 0. 画面の地図

| メニュー | 誰が | 何をする |
|---------|------|---------|
| 受付台帳 | GOLF WING 受付 | 来店したお客様の受付・追記・入会チェック（→ §2〜§5） |
| ダッシュボード | 受付・本部 | 体験→入会率・フィッティング→購入率・会員数の当月実績（→ §6） |
| 体験フォロー | GOLF WING 受付 | 体験から約1週間後のフォロー漏れを防ぐ（→ §6.5） |
| 予約（姫路） | FRANK GOLF | 予約の作成・一覧・お客様Web予約URLの発行（→ §7） |
| FRANK会員 | FRANK GOLF | 会員一覧とプラン管理（→ §8） |
| データ取込 | 本部（古川さん） | Smart Hello のExcelを取り込んでKPIを更新（→ §9） |

## 1. ログインする

1. 上のURLを開く
2. 「メールアドレス または ログインID」と「パスワード」を入れて【ログイン】
   - Shift Cloud（シフトアプリ）と同じID・パスワードです
3. 開けない・「アクセス権がありません」と出る → あなたのアカウントに受付権限（use_reception）が付いていません。**古川さんに「受付権限をください」と伝えてください**

## 2. 朝イチ: タブレットを受付モードにする（常設タブレット）

1. スタッフのPC/スマホでログイン → トップの「一時利用者名簿 — 受付台帳」を開く
2. 【受付URLを発行】を押す → 続けて【受付画面を開く ↗】
3. 出てきた画面（「ご来店受付 — タブレットにご記入をお願いします」）を**タブレットで開いて、そのまま受付カウンターに置く**
   - タブレットで直接ログインする必要はありません。この画面はお客様専用です

## 3. お客様が来たら

- **お客様自身に**タブレットで記入してもらう（お名前・連絡先・利用区分・アンケート・同意・サイン）
- 記入が終わると、スタッフ側の受付台帳に「自己入力済」として自動で並びます
- 書けないお客様・電話受付などは、スタッフが台帳の【＋ 登録】から代わりに入力してOK

## 4. 受付のあと（スタッフの追記）

1. 台帳でそのお客様の行を開く
2. 利用区分（体験利用/フィッティング/打席利用/ビジター打席/その他）・支払・メモを確認して【保存】
3. **入会が決まったら【入会】を押す**（入会率のKPIに自動で反映されます。押し忘れ注意）

## 5. 月次: Excel出力

- 台帳上部の期間を選んで「⬇Excel出力」→ 従来の「一時利用顧客名簿」と同じ形式で出ます（本部提出用）

## 6. ダッシュボード（数字の確認）

- 当月の**体験→入会率 / フィッティング→購入率 / 入会・退会**が出ます。右上の【←】【→】で月を切り替え
- 体験・フィッティングは**受付台帳**から、会員数・退会は**会員名簿（Smart Hello取込）**から集計しています
- 退会は「6月末退会＝6月」に集計。再来者は「割引=再来 / 経路=再来 / 再来日あり」で判定
- 数字が合わないときは、まず受付台帳の入会チェックの押し忘れ（§4-3）を疑ってください

## 6.5. 体験フォロー（GOLF WING）

体験に来て**まだ入会していない方**に、体験から約1週間後に公式LINE等で連絡する画面です。

1. 「体験フォロー」を開く（直近60日・未入会の方が自動で並びます）
2. 「フォロー予定（7日未満）」から順に連絡する
3. 連絡したら**フォロー内容を書いて保存**（例:「公式LINEで再来案内・入会検討中」）
4. 「フォロー対象はありません 🎉」になればその日は完了

## 7. 予約（FRANK GOLF 姫路）

1. 「予約（姫路）」を開く → 日付を選ぶ
2. 電話・店頭の予約は【予約を作成】から入力
3. お客様に自分で取ってもらう場合は【お客様Web予約URLを発行】→ **URLは一度しか表示されません**。QRにして掲示 or HPに載せる（発行し直しは何度でも可）

## 8. FRANK会員

- 「FRANK会員」で会員一覧（レギュラー / 休会 など）とプラン管理
- プランが未登録だと会員に紐付けられません。先に【プランを追加】

## 9. データ取込（本部・古川さん）

Smart Hello から出したExcelを取り込むと、KPIが自動更新されます。

| 取込むもの | 効果 | 注意 |
|-----------|------|------|
| 会員名簿 | 会員数・退会率 | **取込むたびに全件を置き換え**（最新スナップショット。在籍＝スタッフ除く／退会予定は在籍扱い） |
| 予約一覧 | 打席稼働・パーソナル件数 | 期間指定でエクスポート。予約番号で重複排除して**追記** |
| 一時利用者名簿 | 体験→入会率・フィッティング→購入率 | 現行の「（新）一時利用者名簿.xlsx」。取込むたびに移行分を洗い替え（**タブレット受付の実来店は保持**）。日付・氏名のある行のみ |

- 口座番号・クレジットカード等の機微情報は取り込みません
- 取込む前に元のExcelは残しておいてください（やり直せます）

## 10. 困ったとき

| 症状 | 対処 |
|------|------|
| タブレットの受付画面が消えた/変な画面になった | スタッフ画面から【受付URLを発行】→【受付画面を開く ↗】をやり直してタブレットで開き直す |
| 「アクセス権がありません」 | 受付権限（use_reception）が未付与。古川さんへ |
| 保存を押しても反映されない | 通信を確認して画面を再読み込み（それでもダメなら古川さんへ。**紙にメモして後から入力**すれば取りこぼしません） |
| お客様が同意・サインを嫌がる | 同意なしでは受付登録できない運用です。スタッフ判断で無理強いはせず、紙の従来案内へ |

## してはいけないこと

- お客様の行を安易に【削除】しない（間違えて作った行のみ。削除しても復元できるので慌てなくてOK）
- タブレットのお客様画面でスタッフ用URLを開いたままにしない（個人情報が見えてしまいます）
