# RUNBOOK — 契約書の登録と期限管理（Legal OS）現場手順書

対象: 契約書類を扱う担当スタッフ。
URL: https://legal-os-peach.vercel.app

---

## 1. ログインする

- Shift Cloudと同じID・パスワード
- 「アクセス権がありません」→ 法務権限（use_legal または leg_grants）が未付与。古川さんへ

## 2. 契約書を登録する（届いたらすぐ・後回しにしない）

1. 【契約を登録】を開く
2. 入力する項目:
   - **種別**（契約書/覚書/規約/NDA など）
   - **相手方**（会社名・担当者）
   - **契約期間**（開始日・終了日）
   - **自動更新の有無**と**解約通知期限**（「終了の○日前までに通知」の日数。契約書の解約条項に書いてあります）
   - 要点・リスクメモ（わかる範囲でOK。空欄でも登録できます）
3. **PDFや写真をアップロード**（スキャンでも スマホ撮影でもOK。原本は従来どおりファイルへ）
4. 登録すると期限リマインダーが自動で作られ、期限が近づくとCEO AIの判断リストに上がります

## 3. 何を登録する？（判断に迷ったら）

- **登録する**: 契約書・覚書・利用規約・NDA・リース契約・業務委託 → Legal OS（ここ）
- **登録しない**: 請求書・見積書・領収書・レシート → それは経理系（Money OS）。担当が違います
- 迷ったら古川さんに聞くか、とりあえずLegal OSに登録（後で動かせます）

## 4. 一覧・確認

- ダッシュボード: 「90日以内に対応期限の来る契約」が上に出ます — ここが空なら安心
- 一覧: 種別・状態・検索で絞り込み。詳細画面でファイル閲覧・ステータス変更

## 5. 締結・更新・解約の承認について

- **担当スタッフが勝手に締結・解約の判断をしない**こと。正式な承認は古川さんがGENESIS側で行います（Legal OSは保管と期限管理の役割）

## 困ったとき

| 症状 | 対処 |
|------|------|
| アップロードが失敗する | ファイルが大きすぎる可能性（写真を縮小 or PDF化）。再試行してダメなら古川さんへ |
| 解約通知期限がわからない | 契約書の「契約期間」「解約」条項を確認。見つからなければ空欄で登録し、メモに「要確認」と書く |
| 間違えて登録した | 詳細画面から修正できます。削除は論理削除なので復元可能、慌てなくてOK |
