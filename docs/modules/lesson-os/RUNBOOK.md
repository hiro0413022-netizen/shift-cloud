# RUNBOOK — レッスンカルテ（Lesson OS）現場手順書

対象: レッスンを担当するコーチ。
URL: https://lesson-os.vercel.app （ホーム画面に登録しておくと楽です）

---

## 1. ログインする

- Shift Cloudと同じID・パスワード
- 「アクセス権がありません」→ レッスン権限（use_lesson）が未付与。古川さんへ

## 2. レッスンの流れ（1人あたり1〜2分の記録）

1. レッスンノート（生徒一覧）から生徒をタップ（いなければ【＋生徒を追加】— **名前だけでOK**）
2. 「本日のレッスン」タブ →「スイング撮影・登録」→ ファイル選択（**スマホ・タブレットはその場でカメラ撮影**）
3. クラブ・飛距離・メモを入れて【⬆登録】
4. 動画の下に**コーチからのアドバイス**を書いて送信
5. 良いスイングは【☆ベストにする】（生徒ごとに1本）

## 3. 映像を使った指導

- 動画を開くと**描画ツール**が使えます: 直線／円／フリーハンド（4色）・スイングプレーン／前傾ガイドのワンタップ表示・コマ送り・スロー再生（0.25x/0.5x）。【描画を保存】で次回も表示されます
- 「比較再生」タブで**過去と現在**、または**お手本スイングと並べて同時再生**できます

## 4. 進捗（レッスン目標）

- 「進捗」タブで9項目（グリップ〜フィニッシュ）の達成度をスライダーで更新→【登録】
- レーダーチャートは生徒の共有ページにも表示されます。**目標の見える化は継続率に直結**します

## 5. 生徒への共有（アプリ不要）

1. カルテ右上の【🔗生徒へ共有リンク】→ URLをコピー → **LINEで生徒に送る**
2. 生徒はそのURLで自分専用ページ（進捗レーダー・レッスン記録・アドバイス・お手本）をいつでも見られます
3. URLが漏れた場合は【共有を停止する】→ 再発行

## 6. お手本スイング

- 上部メニュー「お手本スイング」でコーチの見本動画を登録。比較再生と生徒ページに表示されます

## 7. データの持ち出し

- 生徒一覧右上の【⬇CSV】でレッスン記録をいつでもエクスポートできます（データは人質になりません）

## してはいけないこと

- 動画を**生徒本人以外に見せない・SNSに上げない**（共有は本人へのURL送付のみ）
- 動画の削除は間違えて撮った場合のみ（上達の記録が消えます）

## 困ったとき

| 症状 | 対処 |
|------|------|
| アップロードが終わらない | 電波を確認。200MB超の動画は分割 or 短く撮り直し |
| 動画が再生できない | 再読み込みして再度【▶再生】（再生リンクは30分で切れる仕様） |
| 共有ページが見られないと言われた | 共有を停止していないか確認→必要なら再発行してURLを送り直す |
| 生徒を間違えて登録した | 古川さんへ（統合・削除できます） |
