## 画面詳細（Phase順）

### Phase 1（管理マスタ）
- A1 スタッフ管理: テーブル（名前・店舗・雇用形態・役職・時給・状態）＋追加/編集ドロワー。店舗変更・時給変更は履歴として保存
- A2 店舗管理 / A3 ブランド管理 / A4 会社設定: シンプルCRUD
- A5 テンプレート管理: 名前・開始/終了時刻（または終日/休みフラグ）・色・適用範囲
- A6 予定種別管理: 名前・色・種別（勤務/休暇/その他）
- A12 監査ログ: 読み取り専用テーブル＋フィルタ

### Phase 2（スタッフ）
- S1 ダッシュボード: 今日のシフトカード → 出退勤状態 → 今日の予定 → お知らせ → 今月勤務時間・給与見込み
- S2 希望提出: 月カレンダー → 日タップ → テンプレートをチップ表示ワンタップ選択＋メモ → 一括提出
- S3 自分のシフト: 月/週表示 / S4 全体シフト: 店舗の日別スタッフ一覧

### Phase 3（シフト作成）
- A7 シフトビルダー: 左に希望一覧、中央に週/月グリッド。希望→ドラッグorタップで割当 → ドラフト保存 → 一括確定(publish) → 通知

### Phase 4（打刻・勤怠）
- K1 キオスク: スタッフグリッド（写真/名前）→ 出勤/退勤/休憩開始/休憩終了の4大ボタン → 完了画面3秒 → 戻る
- A8 勤怠管理: 日別テーブル（予定/打刻/実働/差異バッジ）→ 行クリックで修正ドロワー（理由必須）

### Phase 5（照合・給与）
- A9 月末照合: スタッフ×日のマトリクス。差異セルをハイライト、クリックで詳細
- A10 給与確認: 再認証 → スタッフ別明細（勤務・残業・交通費・手当・控除・見込み）→ CSV出力

### Phase 6（本部）
- H1 本部ダッシュボード: KPI（総人件費・総労働時間・店舗数・アラート数）＋店舗比較テーブル＋人件費率チャート
- H2 AI提案: 一覧（重要度バッジ）→ 詳細 → 承認/却下
