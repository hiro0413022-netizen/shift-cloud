# Workforce OS — SYSTEM

## 目的

人・スタッフ・勤怠・シフト・給与・人件費を管理するドメインOS。MVPは**YOZAN Shift Cloud**。

## 主要ユースケース

| アクター | できること |
|---------|-----------|
| スタッフ | シフト希望提出（テンプレワンタップ＋メモ）、自分/全体シフト確認、ダッシュボード（今日のシフト・勤務時間・給与見込み・お知らせ） |
| 店舗責任者 | スタッフ管理、シフト作成・確定、打刻修正、テンプレート・予定種別管理 |
| iPad（店舗） | 出勤/退勤/休憩開始/休憩終了の打刻 |
| 本部/オーナー | 全店舗俯瞰、月末照合、給与確認・CSV、店舗比較、AI提案の承認 |

## 業務フロー（中核）

```
テンプレ整備 → 希望募集期間オープン → スタッフ希望提出（締切まで）
→ 管理者ドラフト作成 → 確定(publish) → スタッフに通知
→ 当日iPad打刻 → 日次勤怠集計（遅刻/早退/残業/打刻漏れ検知）
→ 打刻修正（理由必須・履歴保存）→ 月末照合（希望/確定/打刻/実勤務/給与）
→ 給与見込み確定 → CSV出力
```

## 照合の例

予定 11:00–20:00 / 打刻 11:07–20:18 → 実働 8時間11分（休憩除く）/ 差異: 遅刻7分・退勤18分超過

## 境界

- 給与の**計算と見込み表示・CSV出力**まで。振込・年末調整・社保は対象外（将来の会計・給与モジュール連携）
- 予約数連動の人員提案はAI提案の受け皿のみ（予約OS連携後）
