# Workforce OS — MVP_SCOPE

## 含む

- 認証（メール＋パスワード / ログインID方式）・権限（ロール×スコープ）
- 会社・ブランド・店舗・スタッフ管理（複数店舗兼務、時給履歴）
- シフトテンプレート・予定種別（会社ごと自由編集）
- シフト希望提出（テンプレワンタップ＋メモ、募集期間管理）
- シフト作成（ドラフト→確定→通知）、自分/全体シフト閲覧
- iPad打刻（出勤/退勤/休憩、端末トークン認証）
- 勤怠集計（実働・遅刻・早退・残業・打刻漏れ）、打刻修正（理由・履歴）
- 月末照合、給与見込み（時給ベース）、CSV出力、給与画面の再認証保護
- お知らせ・店舗イベント・アプリ内通知
- 監査ログ、AI提案の保存・承認構造
- 本部ダッシュボード（人件費・店舗比較・アラートの基本）

## 含まない（構造のみ用意）

- 打刻の顔認証・位置情報・IP検証・QR・不正検知
- 有給・交通費の申請フローUI（approval_requestsテーブルのみ）
- LINE/Slack実送信、LINEログイン
- 予約・CRM・在庫・会計・レッスン管理連携
- 売上データ入力（本部の人件費**率**は売上連携後）
- 給与振込・社保・年末調整
- SaaS課金・セルフサインアップ
- AI提案の自動生成（Phase 6でルールベースの最小版のみ）

## MVP完了の定義

GOLF WING 4店舗（宝塚・夙川・明石・新宿）で、希望提出→シフト確定→打刻→月末照合→給与CSVが1サイクル回ること。
