## 業務ルール

### シフト希望
- スタッフは時間を手入力しない。管理者作成のテンプレートをワンタップ選択
- メモ欄あり（例:「17時以降なら残れます」「宝塚勤務希望」）
- 募集期間（対象月・締切）は管理者が設定

### 雇用形態と給与
- `employment_type`: 社員 / アルバイト / 業務委託 / レッスンプロ
- 時給は履歴管理（`staff_wages`、有効開始日つき）。昇給しても過去の給与計算は不変
- 給与見込み = 実働分 × 時給 + 交通費 + 手当 − 控除（MVPは時給ベース。月給者は表示のみ）
- 丸め: 実働1分単位記録。計算時丸めは会社設定（デフォルトなし）

### 勤怠
- 遅刻 = 確定シフト開始より打刻が遅い / 早退 = 終了より早い / 残業 = 終了超過
- 打刻漏れ = シフトあり＆打刻ペア不成立 → アラート
- 修正は理由必須、修正前後と修正者を保存、元打刻は不変

### 予定種別（初期セット例）
現場 / レッスン / ラウンドレッスン / 会議 / 営業 / 撮影 / 出張 / 休日 / 有給 / 研修 / VIP対応 / イベント / コンペ — 会社ごとに自由に追加可

### シフトテンプレート（初期セット例）
早番 / 遅番 / 終日 / 午前 / 午後 / レッスン / 工房 / 受付 / ラウンド / 休み — 店舗またはブランド単位で管理者が自由に作成

### 店舗イベント共有
設備点検 / ミーティング / キャンペーン / VIP来店 / コンペ / 閉店時間変更 / 社内研修 — 店舗全体に表示
